# Heart-Disease-Prediction-ML
 
 
 ## Project Overview
This project predicts whether a patient is likely to have heart disease using Machine Learning.

The project includes:
* Data Cleaning
* Exploratory Data Analysis (EDA)
* Data Visualization
* Logistic Regression Model
* Model Evaluation
* Feature Importance Analysis

## Dataset
Dataset: Heart_Disease_Prediction.csv

The dataset contains medical information such as:
* Age
* Cholesterol
* Blood Pressure
* Heart Rate
* Other Health Parameters

Target Variable:
* Heart Disease
  
## Technologies Used
* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Project Workflow
### 1. Data Collection
Dataset loaded using Pandas.

### 2. Data Cleaning
* Checked missing values
* Removed duplicate records

### 3. Exploratory Data Analysis

Visualizations performed:
* Age Distribution
* Heart Disease Count
* Cholesterol Distribution
* Correlation Heatmap
* Confusion Matrix

### 4. Machine Learning Model
Algorithm Used:
* Logistic Regression

### 5. Model Evaluation
Metrics Used:
* Accuracy Score
* Classification Report
* Confusion MatrIX
  
## Results
The Logistic Regression model was trained and evaluated successfully.
The model predicts the presence of heart disease based on patient health information.
